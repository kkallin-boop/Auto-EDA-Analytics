Dashboard
Fitur dan proses inti dari Auto EDA Analytics
Auto EDA Analytics
Platform Analisis Data Cerdas berbasis web yang dibangun menggunakan Flask sebagai backend server dan Python sebagai inti pemrosesan data, serta HTML, CSS, dan JavaScript untuk antarmuka pengguna. Sistem dirancang untuk mengotomatisasi proses Exploratory Data Analysis (EDA), mulai dari unggah dataset, data cleaning, analisis statistik, visualisasi, hingga ekspor hasil dalam satu dashboard terintegrasi.
Flask (Backend) Python 3 / pandas / NumPy Matplotlib / Seaborn Chart.js 4.4 PapaParse 5.4 SheetJS (XLSX) SciPy
Upload Dataset
Mendukung unggahan dataset dalam format CSV, Excel, dan TXT. Setelah file dipilih, sistem akan membaca data dan mengidentifikasi tipe setiap kolom secara otomatis sebagai langkah awal proses analisis.
Data Preview
Menampilkan isi dataset beserta informasi dasar mengenai kualitas data. Fitur ini membantu pengguna melakukan pemeriksaan awal sebelum melanjutkan ke tahap analisis berikutnya.
Data Cleaning
Membantu meningkatkan kualitas dataset melalui penanganan missing value, penghapusan data duplikat, dan standarisasi data kategorikal sehingga data lebih siap untuk dianalisis.
Descriptive Statistics
Menyajikan ringkasan statistik secara otomatis untuk membantu pengguna memahami karakteristik data, baik untuk variabel numerik maupun kategorikal.
Visualization
Menyediakan berbagai visualisasi seperti histogram, boxplot, scatter plot, pie chart, dan heatmap korelasi untuk membantu proses eksplorasi data secara lebih intuitif.
Time Series
Digunakan untuk menganalisis dataset yang memiliki variabel waktu dengan menampilkan tren data, moving average, dan pola perubahan dari waktu ke waktu.
Download Results
Memungkinkan pengguna menyimpan dataset yang telah diproses maupun hasil analisis ke dalam beberapa format file untuk dokumentasi atau penggunaan lanjutan.
